div는 의미가 없는 단순한 영역을 나타나게 하는 박스를 의미한다.

span 요소는 의미가 없는 단순한 내용의 그룹

시멘틱 구조는 각 영역별로 그 의미를 가지고 쓰이도록 처리된 영역

header, nav, section, article, aside, main, footer, figure

영역을 나타내는 의미를 구조

시멘틱 구조에 대해 알아보기

header : 웹페이지의 상단에 존재, 웹의 로고/h1을 담으면서 전체 웹페이지 구조를 파악 할 수 있는 nav영역을 함께 포함시키는 기능을 가진다.

navigation

nav : 웹페이지 내부에서 클릭하면 해당하는 페이지로 이동하게 만들어주는 링크들의 집합/ navigation요소를 담아놓은 영역

하나의 대표 내용을 보여주는 영역

section : 하나의 대표 내용을 보여주는 영역으로 다른 내용과 연계가 되어있는 부분, 본문 : 광고/banner

다른내용과 연계가 되지 않아도 별도의 독립영역

article : section과 유사한 기능을 가진 영역으로 본문 내용, 다른 내용과 연계가 되지 않아도 별도의 독립영역으로 설정해도 문제없는 영역 : 게시판 안내/신문기사/광고

another 영역

aside : another 영역을 지칭하는 것으로, 사이트의 구성요소에 없어도 전혀 문제되지 않는 영역, 있으면 도움이 되는 영역

main : section, article의 영역을 정리해서 더 큰 영역으로 담는 곳 아직 대부분의 사이트에서는 완벽하게 지원하지 못하고있다.

footer는 회사/사이트의 정보를 담고 있는 영역

footer : 이름으로 해석하면 하단에 존재한다는 의미를 가지고 있지만, 꼭 하단에만 존재하지는 않는다.
footer는 회사/사이트의 정보를 담고 있는 영역을 지칭 주소, 연락처, 제작연도, 지침, family site
정보가 보이지 않는다고 해도 사이트에 대한 내용은 반드시 존재 해야한다.
