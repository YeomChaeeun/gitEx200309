canvas 곡선그리기

곡선은 표현 메소드가 4가지 이다.

arc, arcTo, quardraticCurveTo, bezierCurveTo

  • arc : x,y,지름,시작점,끝점,방향(시계/반시계)
  • arcTo : moveTo, lineTo와 함께 사용하는 형태로, 두개의 선의 모서리 곡선형태
  • quardraticCurveTo : 베지어곡선 포인트 1개를 이용하여 그리는 곡선형태
  • bezierCurveTo : 베지어곡선 포인트 2개를 이용하여 그리는 곡선형태

현재 브라우저는 canvas 기능이 지원되지 않는 브라우저 입니다.
최신 브라우저를 이용하세요.
