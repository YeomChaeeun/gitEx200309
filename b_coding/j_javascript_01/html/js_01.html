javascript의 개념이해하기

javascript는 제어의 기능을 하는 언어 : html/css문서의 내용을 삽입/삭제/수정
서버를 구성하여 처리하기, 동기화처리, animation/button클릭시 처리되는 기능

javascript는 java언어와는 전혀 다른 언어로 기능부터 형태, 역할 모든것이 다르다.

다양한 버전과 다양한 스크립트 언어들이 존재
종류 : javascript, coffeescript, actionscript, typescript, dart, ecmascript ...
버전 : ecma를 기반으로 다양한 버전이 존재 es5 / es6 이상


javascript 문서 사용하는 위치 파악하기

  • 1. head내부에 작성
  • 2. body내부 문서 중간에 작성
  • 3. body내부 끝에 작성

  • 5. 외부에서 불러오는 방법(body에서 작성)
